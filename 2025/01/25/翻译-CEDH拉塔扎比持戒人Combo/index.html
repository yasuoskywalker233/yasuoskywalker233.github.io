翻译-CEDH拉塔扎比持戒人Combo

moxfield卡组：

卡组简介：

魔戒：中洲传说的魔戒引诱效果和乌尔博格的拉塔扎比有着独一无二的联动：带有死去时魔戒引诱你的传奇生物（勇敢的山姆怀斯、耐心谋策鬼咕噜、白塔守护波洛米尔）死去时，拉塔扎比会派出一个非传奇的token，同时魔戒引诱你的异能会在堆叠之中。结算并选择token作为持戒人将其变为传奇生物，牺牲token触发拉塔扎比和魔戒引诱来达成无限循环，从而制胜。拉塔扎比有守护异能提供自我保护、3/3的合格身材以及警戒足以在生物攻防中应对自如。

套牌选择

为什么选择这个套牌？

此套牌利用白色的锁效应使得整个游戏降速，并配合黑色的导师效应来找齐combo组件。combo组件之间有极其优秀的配合、多路线多维度制胜。作为生物套牌，能避开大部分常用的康。这套牌在面对各种常见锁时都能运转并且获胜，包括神器Hate、墓地Hate、依法治理效应等。如果你喜欢生物套牌、喜欢有主动的游戏计划，请务必尝试这一套牌。

为什么不选择这个套牌？

就像许多非织命的黑白套牌，这套牌在手牌资源上不占优势。并且除开个别卡，整套牌在堆叠上的互动能力比较孱弱，面对启动速度更快的卡组也是个问题。面对锁牺牲效应（如执拗雄野牙刹）或进坟替代效应（道西虚空行者）时比较棘手，但去除咒语能够应对。如果你喜欢抓很多牌，在堆叠上互动，或者采取被动的策略，请不要选择这套牌。

Combo解析

魔戒引诱循环

这些魔戒引诱循环是利用主将异能的主要方式。他们能带来无限ETB、无限LTB、无限牺牲效应。魔戒引诱循环和其他组件配合能达成无限费、无限伤害、磨牌制胜等制胜手段。

主将+魔戒引诱生物+牺牲源循环

  • 主将：乌尔博格的拉塔扎比
  • 魔戒引诱生物：勇敢的山姆怀斯/耐心谋策鬼咕噜/那兹古尔（作为持戒人）
  • 牺牲源

步骤：

  1. 牺牲魔戒引诱生物
  2. 如果牺牲的是咕噜，先结算主将的异能派出非传奇token，再结算咕噜的离场异能选择token作为持戒人
  3. 如果是山姆怀斯或那兹古尔，先结算主将异能派出token，token触发ETB魔戒引诱选择token作为持戒人
  4. 循环

主将+白塔守护波落米尔循环

  • 主将
  • 白塔守护波洛米尔

步骤：

  1. 启动白塔守护波洛米尔的异能，牺牲他
  2. 主将异能触发，先结算主将异能派出token
  3. 再结算白塔守护波洛米尔的异能，选择token作为持戒人
  4. 循环

制胜

在达成魔戒引诱循环后，有多种制胜路线：

  1. 牺牲源制胜：爆破工段或痴呆祭坛
  2. 嘬血效应：牟利镇长坎拔等

无限费

在达成魔戒引诱循环后，即使没有嘬血效应，也能通过达成无限费制胜

  • 在魔戒引诱循环中，使用非瑞克西亚祭坛或繁殖地易魂师作为牺牲源
  • 在进行新一轮魔戒引诱循环前，使用传承遗宝横置持戒人产费
  • 使用养子尸变组合技，云移产费神器达成无限费，或用繁殖地易魂师牺牲1/1token

在达成无限费后，

  • 释放超大的自行弩炮
  • 释放超大的异尘余生（最新的牌表已经没有这张卡了）
  • 山姆怀斯莲花瓣组合技（下方）

山姆怀斯-莲花瓣组合技

当我们在魔戒引诱循环中使用勇敢的山姆怀斯作为循环生物时，我们可以利用山姆怀斯的异能反复捞取莲花瓣，然后通过盲目服从进行嘬血制胜

  • 主将、勇敢的山姆怀斯、莲花瓣、盲目服从在场
  • 我们能够启动魔戒引诱循环

步骤：

  1. 牺牲莲花瓣产一白费或黑费
  2. 启动魔戒引诱循环，当勇敢的山姆怀斯token进场时，捞取莲花瓣回手
  3. 释放莲花瓣，触发盲目服从的敲诈异能
  4. 用第1步产的费支付榨取异能的费用进行嘬血
  5. 重复

注意：该组合技存在施放咒语的动作，会受到琉晶研究和神秘鱼阵的影响！

其他循环

传奇规则循环

当某些时候我们无法通过牺牲效应（通常是没有抽到牺牲源，或被锁住了）达成魔戒引诱循环的时候，可以通过传奇规则强行启动魔戒引诱循环

  • 主将、勇敢的山姆怀斯、嘬血效应在场
  • 有1W来释放勇敢的山姆怀斯
  • 能够消灭/牺牲魔戒引诱生物的手段和足够的费用

步骤：

  1. 牺牲或消灭山姆怀斯
  2. 主将异能触发，派出非传奇token
  3. 山姆怀斯token进场，成为持戒人，同时将勇敢的山姆怀斯捞回手上
  4. 释放勇敢的山姆怀斯
  5. 传奇规则触发，牺牲山姆怀斯token
  6. 主将异能触发，派出非传奇的山姆怀斯token，同时选其成为持戒人
  7. 循环步骤5~步骤6
  8. 期间，嘬血效应会触发从而制胜

勇敢的山姆怀斯可以换成耐心谋策鬼咕噜：

  1. 牺牲或消灭咕噜
  2. 先结算主将异能触发，派出非传奇token
  3. 再结算咕噜的离场异能，非传奇咕噜token成为持戒人
  4. 支付B，牺牲咕噜token启动坟墓场里咕噜的异能，将其移回手上
  5. 咕噜token离场时同样会触发主将的异能和自身的离场异能，我们重复步骤2~3再派出一个咕噜token作为持戒人
  6. 支付1B释放耐心谋策鬼咕噜
  7. 传奇规则触发，牺牲咕噜token
  8. 咕噜token离场时同样会触发主将的异能和自身的离场异能，我们重复步骤2~3派出咕噜token作为持戒人
  9. 传奇规则再次触发，我们进入步骤7~8的循环
  10. 期间，嘬血效应会触发从而制胜

谢迪西路线

  • 主将、其他一个生物在场
  • 我们拥有3BBB
  • 不死宰相谢迪西在手上
  • 我们的生命值大于5

步骤：

  1. 释放不死宰相谢迪西
  2. 谢迪西的榨取异能触发，牺牲其他生物找牺牲
  3. 支付B释放牺牲，目标谢迪西，获得BBBBB
  4. 主将异能触发派出谢迪西token，谢迪西榨取异能牺牲自己找活尸法
  5. 支付B释放活尸法，捞取谢迪西进场，榨取异能牺牲自己找痴呆祭坛，余BBBB
  6. 主将异能触发，派出谢迪西token，进场牺牲自己找耐心谋策鬼咕噜，余BBBB
  7. 剩下的BBBB刚好可以释放痴呆祭坛和咕噜，启动魔戒引诱循环磨牌制胜

碎响骨王奥索兹路线（暂未出现在牌表里）

  • 主将、碎响骨王奥索兹、非瑞克西亚祭坛或繁殖地易魂师在场

步骤：

  1. 牺牲碎响骨王奥索兹，加B或派出1个珍宝
  2. 主将异能触发派出非传奇的奥索兹，奥索兹异能触发派出传奇乱骨
  3. 牺牲传奇乱骨，加B或派出1珍宝，乱骨异能触发将奥索兹捞回手
  4. 主将异能触发派出非传奇乱骨
  5. 牺牲非传奇奥索兹，加B或派出1珍宝，奥索兹异能触发派出传奇乱骨
  6. 牺牲传奇乱骨，加B或派出1珍宝，主将异能触发派出非传奇乱骨
  7. 通过上述步骤的产费或珍宝再次释放奥索兹，此时余下B或1珍宝，场上还有1个非传奇的乱骨
  8. 循环，得到无限费和无限2/1非传奇乱骨token

没主将时的一些路线

  • 日冠赫利欧德+自行弩炮
  • 掠盗飞禽凡希科+阿加莎的灵魂鼎镬（简称锅），锅放逐自行弩炮给凡希科加豆
  • 养子尸变组合技
  • 墓场匍尸组合技，主将创造的token是灵俑
  • 非瑞克西亚祭坛或繁殖地易魂师 + 勇敢的山姆怀斯或梦巢猫魇路拉斯 + 卡娅的魂体（这个combo不在当前的牌表中）

一些单卡解析

魔戒引诱生物

  • 勇敢的山姆怀斯：闪现异能很重要，能让你在瞬间时机使另一个生物成为持戒人，配合主将的异能使生物“躲过”去除咒语
  • 耐心谋策鬼咕噜：异能可以从坟墓场爬回手上，使得咕噜成为入土的主要目标
  • 白塔守护波洛米尔：能够自我牺牲，使得波洛米尔无需牺牲源即可启动魔戒引诱循环，配合传承遗宝可以达成无限费。波洛米尔在场上时可以针对免费咒语和替费咒语
  • 那兹古尔：注意，那兹古尔本身不是传奇，需要在那兹古尔进场时，先选择自己作为持戒人

嘬血效应单卡

  • 虐之华寇族黜人依拉丝：达成魔戒引诱循环时，可以通过嘬血制胜
  • 肉钩大屠杀：带有扫场功能的嘬血单卡
  • 筑拉波割喉客：高效的嘬血单卡
  • 牟利镇长坎拔：另一张传奇嘬血单卡，还可以白嫖对手的token

牺牲源

  • 痴呆祭坛：能够磨牌制胜，也能自磨耐心谋策鬼咕噜进坟，然后用咕噜的异能捞回手上
  • 爆破工段：配合魔戒引诱异能可以扎针制胜
  • 非瑞克西亚祭坛：配合魔戒引诱循环可以产无限费，也可以达成墓场匍尸组合技
  • 繁殖地易魂师：和非瑞克西亚祭坛用法差不多，但注意要有嘬血效应来保证你的生命值
  • 索蓝医师约格莫夫：集牺牲源、抓牌、去除于一体，但注意我们使用约格的目的是找到嘬血单卡来抵消约格启动异能时支付的生命值

其他一些可以考虑携带的牺牲源：

  • 拽脏预言师、腐食怪：能被亿欧护林队长找到
  • 巴洛梅德普西迪奥：一个传奇牺牲源
  • 秽血拉札柯：掘坟效应或者阿加莎的灵魂鼎镬的绝佳目标

其他combo组件

  • 自行弩炮：达成无限费后的终结手段
  • 日冠赫利欧德：配合弩炮
  • 阿加莎的灵魂鼎镬：配合掠盗飞禽凡希科和自行弩炮，本身放逐道西虚空行者、亿欧护林队长也极其强力
  • 掠盗飞禽凡希科：配合阿加莎的灵魂鼎镬和自行弩炮，本身自带飞行穿透效果也可以配合织命使堤谟娜
  • 葛力安养子阿波戴尔亚德恩：经典养子尸变组合技
  • 墓场匍尸：配合非瑞克西亚祭坛可以达成经典的无限费组合技，配合嘬血效应制胜

其他可以考虑携带的组件：

  • 狮族遗宝护卫：另一个启动养子组合技的组件
  • 一锯两断：当我们主将在场时，这是一张可以创造许多ETB/死去效应的单卡，或者启动传奇规则循环。澄穹猗旺是绝佳目标
  • 恶念重返：和一锯两断类似，可以启动传奇规则循环，但法术时机稍显笨重
  • 澄穹猗旺：一张可以帮助我们快速找齐组件的单卡，和一锯两断有绝佳的配合
  • 异尘余生：无限费出口
  • 卡娅的魂体：非瑞克西亚祭坛或繁殖地易魂师 + 勇敢的山姆怀斯或梦巢猫魇路拉斯 + 卡娅的魂体（这个combo不在当前的牌表中）
  • 碎响骨王奥索兹：碎响骨王奥索兹路线（暂未出现在牌表里）

法术力源

  • 魔法力库、阳光戒、五彩玛珂、玛珂钻石、莲花瓣：正向产费神器，莲花瓣还可以达成山姆怀斯-莲花瓣组合技
  • 琥珀玛珂：我们的套牌有许多传奇生物
  • 蛋白玛珂：虽然我们套牌的神器数量有点少，但由于缺乏其他选择，这张卡是必要的免费加速神器
  • 传承遗宝：无限费组件之一，套牌有许多传奇生物也和这张卡有良好的配合
  • 秘法印记、阶层饰符：传统加速神器
  • 黑暗祭礼、柯帮祭礼：能够加速帮我们速出主将、至尊戒或其他组件
  • 牲祭弱者：套牌生物密度高，足够支持牲祭弱者的额外费用，也可以用来启动传奇规则循环
  • 牺牲：一般情况下比牲祭弱者差，但能启动谢迪西路线
  • 贪污夏警洛索：本身是个传奇生物，cedh桌上的常客
  • 税赋压身：码头哥死后，税赋压身变得值得加入套牌（虽然你一个黑白色组也带不了码头）

功能性单卡

  • 艾斯波哨卫：沾白CEDH必带
  • 欧格玛的档案师：这个套牌很需要抓牌引擎！
  • 织命使堤谟娜：我们不那她当主将，不代表我们不将她加入99之中
  • 觉醒梦魇布蕾德：能够牺牲掉自己的一些锁，也能提供一定的去除、抓牌效应
  • 至尊戒：抛瓦！
  • 祸事成双：一张能帮我们每轮抓很多牌的新卡

导师

  • 邪魔导师、吸血鬼导师、玉玺：黑色必带导师
  • 入土：找耐心谋策鬼咕噜，再通过其异能拿回手上启动魔戒引诱循环，或者找自行弩炮配合阿加莎的灵魂鼎镬达成组合技
  • 启蒙导师：可以找爆破工段、痴呆祭坛等神器来制胜，也能找日冠赫利欧德、尸变等卡
  • 恶魔意图：和牲祭弱者作用差不多，但是有导师效果
  • 愿爪饰符：在有依法治理效应的情况下，能让我们释放找到的牌
  • 腐败协定：能在瞬间时机找我们任何想要的单卡，但要放逐卡牌
  • 不死宰相谢迪西：谢迪西导师链条
  • 唤生挽歌：集入土+回收效应的单卡

锁

  • 卓尼斯官员、道西虚空行者、反对派密探：必带
  • 盲目服从：横进效应能瘫痪一些神器组合技或生物组合技，还是山姆怀斯-莲花瓣组合技的制胜
  • 止咒高僧：保护我们的combo
  • 奥克弓手：听说你喜欢带琉晶研究，听说你喜欢抓牌？

其他可以考虑携带的锁：

  • 震撼寂静：套牌生物密度比较高，可以考虑携带
  • 哲娜与琳法拉：季宁、西塞、屈东杀手

去除与互动

  • 化剑为犁：经典单卡
  • 恶意眩闪：替费咒语，同时可以帮助我们启动传奇规则循环
  • 幽寂：替费的化剑为犁
  • 异灯行进：灵活的去除
  • 封口：保护combo，或阻止对手的combo
  • 亿欧护林队长：提供保护，同时能找艾斯波哨卫、墓场匍尸、自行弩炮

其他一些可以考虑携带的去除与互动：

  • 毒雨倾盆：在生物套牌较多的环境里很好用
  • 天使恩典、刚毅眩闪：你这回合不许赢！刚毅眩闪同时可以帮助我们启动传奇规则循环

掘坟效应

  • 活尸法：帮我们快速启动传奇规则循环，谢迪西组合技链条的组件，捞对手的怪
  • 尸变、死灵术：养子尸变组合技
  • 瑟纹的复还：可以回收我们大部分的组合技组件

其他一些可以考虑携带的掘坟组件：

  • 亡者之舞：养子尸变组合技组件
  • 梦巢猫魇路拉斯：能捞坟，还是传奇，有非瑞克西亚祭坛或繁殖地易魂师 + 勇敢的山姆怀斯或梦巢猫魇路拉斯 + 卡娅的魂体（这个combo不在当前的牌表中）组合技，但是速度比较慢

一些地牌

  • 巫祭附魔师：灵活的地，还可以喂给幽寂、异灯行进等，同时还可以去除一些结界
  • 腐魔突堕：灵活的地
  • 群英广场：我们的传奇生物密度足够，并且可以提供一定的保护
  • 非瑞克西亚高塔、拉佐特采石场：能够帮我们启动传奇规则循环

其他一些可以考虑携带的地

  • 克撒传：找族群祭坛

一些没有考虑带上的卡

  • 倒胃口：套牌平均法术力值有点高（1.56），不太适合
  • 死冥权能：三点黑太深了
  • 魔判官之攫：无法真正制胜
  • 波拉斯尊殿：需要配合养子尸变组合技才能获胜
  • 恶魔协商：我们可以带多张那兹古尔，然后用恶魔协商找到它。但这样我们就要放弃其他路线并且更依赖主将
  • 吉拉波之望：很好的保护，但在第一回合之后抓到它就有点鸡肋
  • 叛逃虫械斯勒夫：很好的保护，但如果不是起手抓到的话就有点鸡肋

游戏计划与套牌操作指南

整体计划

尽管有一些锁牌，但我们希望尽快赢得对局，而不希望维持僵局。所以启动组合技是我们的首要任务。对手是什么套牌不是很关键，但我们在牌桌的位次比较重要。

在1号位，我们需要一个尽可能贪、尽可能快的起手，目标是在对手建立起反制手段之前赢下对局。在2号和3号位，我们需要一个稍微慢一点的计划，希望能在对手逐渐消耗资源应对其他威胁时，顶着对手进行计划。在4号位，我们又需要一个比较贪的计划来弥补位次带来的劣势。

起手应当包含一些加速，一个导师或者赚牌引擎，和一个组合技组件。基于组合技组件，你可以建立一条获胜路线。理想情况下，我们使用封口效应来保护组合技。加速组件是最重要的，应当尽可能调度寻找加速组件或通过导师来寻找加速组件。如果一个起手没有包含组合技组件，但有导师，你可以搜寻一些加速组件或赚牌引擎。
